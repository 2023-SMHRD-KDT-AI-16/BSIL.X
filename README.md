# BSIL.X
BSIL.X 핵심 프로젝트입니다.

1.프로젝트명(팀명:BSIL.X)

2.서비스소개

  바쁜 직장인들 및 현대인들을 위해 고안한 서비스로 간단하게 재료의 가격과 도시락 레시피를
  한눈에 쉽게 볼 수 있습니다. 
  
3.프로젝트기간

  2024.02.23~ 2024.4.5
  
4.주요기능
  • 가격 정보를 업데이트하여 대형마트 및 전통시장의 식재료 가격 변동 현황을 시각화
  • 선택한 레시피의 식재료 가격을 합산하여 총 비용을 계산하고 제공
  • 만든 도시락의 가격과 외식 비용을 비교하는 그래프 제공으로 경제적인 선택 상승
